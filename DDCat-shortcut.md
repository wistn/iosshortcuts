# DDCat多多猫快捷按钮v1.1 @Guang
## 功能
（多多猫辅助）对当前网页调用多多猫打开，如果当前网站是多多猫插件对象就自动进入多多猫插件。（安卓版H5浏览器扩展插件中心已有同名插件实现同样功能）
## 使用
在浏览器/其他app选择发送/分享，弹出的菜单找快捷指令（没看到就进入更多来开启），点击指令DDCat多多猫快捷按钮（第一次遇到同意提示就确认）。
